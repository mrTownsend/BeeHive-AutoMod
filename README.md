# Installation

*[p] is your prefix.*

* Make sure Downloader is loaded:<br>
  `[p]load downloader`

* Add the repo to your bot:<br>
  `[p]repo add BeeHive-AutoMod https://github.com/mrTownsend/BeeHive-AutoMod`

* Install the cog:<br>
  `[p]cog install BeeHive-AutoMod automod`

* Load automod:<br>
  ``[p]load automod``
